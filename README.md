Proyecto Freelance: Software de gestion para el estacionamiento El AlFARERO para la gestión vehicular que automatiza el cálculo de tarifas mediante un sistema de temporizador en tiempo real. El software permite el registro detallado de unidades (placas, propietarios, características y observaciones), además de incluir un módulo de auditoría histórica para el control de ingresos y flujo operativo diario.
